Kjøreflyt

launchd starter sjekk-og-kjoer.sh hvert 30. min (hverdager)
avbryter stille hvis alle 5 resultatfiler finnes for dagen
run-rapport-lokalt.sh kjører alle 5 tester i rekkefølge
npm run arkiv kopierer resultater til docs/ og genererer denne siden
git push publiserer til GitHub Pages
Alle tester importerer START_URL og grenseverdier fra config.js.
Resultater lagres i rapporter/YYYY-MM-DD/ og arkiveres under docs/arkiv/.
